# Escenario: Análisis de ventas

El proyecto consiste en el análisis de ventas de productos identificados mediante:
ID, utilizando un dataset anual para obtener indicadores básicos y representar información de manera gráfica.

---------------------------------------------

# Objetivo del proyecto:

Desarrollar un sistema simple capaz de crear archivos y analizar ventas utilizando las siguientes herramientas:

- Python
- Git
- GitHub
- Jira
- Google Colab

El objetivo principal es obtener indicadores básicos de ventas y representar la información mediante gráficos simples.

---------------------------------------------

# Dataset utilizado:

https://gist.github.com/khanusama20/ee33c2869dd5cf3cebdf020be1ca43f6#file-sales_sample_2024-csv

El dataset contiene registros comprendidos entre el 1 de enero y el 31 de diciembre de 2024.

# Variables del dataset:

Variable - Descripción

ID - Identificador del producto 
fecha_venta - Fecha en la que se realizó la venta 
ventas - Cantidad de unidades vendidas 

---------------------------------------------

# Funcionalidades implementadas:

- Lectura de archivos CSV
- Cálculo de ventas totales
- Identificación del producto más vendido
- Generación de gráficos
- Generación de resultados básicos
